# BRICK BREAKER

Project 3 for C220
3/3/2023

I have made this classic brick breaker game very 'juicy'


## Implementation
1. change colors
2. squeeze and stretch
3. animate elements when hit
4. sound effects
5. music
6. animate bricks
7. particles
8. fall off screen
9. comet trail
10. eyes
11. eyes follow action
12. mouth
13. shaders
